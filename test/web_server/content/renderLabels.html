Test Page

All of the modules exported methods are exposed on this page under window.labelUtils. Wrapped versions of each function that include drawing bounding boxes and adding offsets are exposed on window.testFixtures.

To render somwthing, open dev tools > console, then execute one of the following:

  • testFixture.resetSvgContents()
  • horizontal single text label: testFixture.getSingleLineLabelDimensions({text: "Label 1",fontSize:"20", fontFamily:"Arial",fontWeight:"lighter",orientation:"HORIZONTAL", offset:{x:0,y:50}})
  • vertical single text label: testFixture.getSingleLineLabelDimensions({text: "Label 1",fontSize:"20", fontFamily:"Arial",fontWeight:"lighter",orientation:"TOP_TO_BOTTOM", offset:{x:0,y:50}})
  • render horizontal wrapped label testFixture.addLabel({text: "lorem ipsum blah blah foo banana rama", offset:{x:50,y:50}, bounds:{width:100,height:100}})
  • render vertical wrapped label testFixture.addLabel({ orientation: "TOP_TO_BOTTOM", text: "lorem ipsum blah blah foo banana rama", offset:{x:50,y:50}, bounds:{width:100,height:100}})
